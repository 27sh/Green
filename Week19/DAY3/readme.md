# 람다식 문법

매개변수가 하나이면 자료형과 소괄호를 생략할 수 있다.

총괄호 안의 구현부가 한 문장이면 중괄호를 생략할 수 있다.

중괄호 안의 구현부가 한 문장이라도 return 문이 있다면 중괄호를 생략할 수 없다.

매개변수가 두 개 이상이면 소괄호를 생략할 수 없다.

중괄호 안의 구현부가 반환문 하나라면 return과 중괄호 모두 생략할 수 있다.
